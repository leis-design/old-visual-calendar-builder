Want to build a visual calendar for your course?
Use this tool to build your visual calendar and easily transfer it to eClass.
Creating your calendar:
  1. Click the tile next to the week, under the day where you would like to add a course event
  2. Select the type of event you want to add by clicking on the "event type" drop down menu
  3. If you want the event to repeat weekly change the option from "Don't Repeat" to "Repeat Weekly" in the "recurring event" drop down menu
  4. Click the "Save" button to add your event

Note: If you refresh this page while you are building the calendar, you will lose your progress.

  1. Click the tile next to the week, under the day where you would like to delete a course event
  2. Select the course event you wish to delete from the ‘event type’ drop down menu
  3. Click on “Delete selected event”

Note: You can use the “Delete all” button to clear all events for the selected day.

Transfering your visual calendar to eClass:

You can display this visual calendar anywhere in your eClass course.

If you are using Lassonde’s eClass template, you can display your visual calendar in the course schedule page.

You can paste the HTML code anywhere in eclass by toggling to the html view in the text editor

  1. In the eClass text editor, click on the drop down arrow to expand your editing options
  2. Switch to HTML mode
  3. Paste in your code
  4. Click "Save and display"
